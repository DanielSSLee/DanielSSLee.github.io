Bio

Hi, I’m Daniel, a software engineer focused on building systems from the ground up.

I enjoy working across the stack, from low-level C++ graphics and rendering engines to full-stack web applications and data-driven tools. I’ve implemented custom ray tracers, OBJ mesh pipelines, LiDAR-based scanning systems, and scalable web backends using PostgreSQL and Express.

I’m particularly interested in systems design, performance optimization, and building software that is both technically rigorous and practical. I approach engineering as a craft, prioritizing clarity, structure, and long-term maintainability.
